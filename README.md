# weapon

Weaponized packages?!

## Usage

```js
var weapon = require('weapon');

console.log( weapon(0) ); // 0
console.log( weapon(1) ); // 1

// special feature: also works with non-numeric input
console.log( weapon('s') ); // s
```

## Installation

With [npm](http://npmjs.org) do:

```bash
$ npm install weapon
```
